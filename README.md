# SansNetwork

[![Build Status](https://travis-ci.org/sanscentral/sansnetwork.svg?branch=master)](https://travis-ci.org/sanscentral/sansnetwork)

SansNetwork is a golang library for direct Bitcoin protocol interaction

## Build

Requires Go version 1.9 or later. Use [go dep](https://github.com/golang/dep) for dependency management

Run all unit tests with `$ go test ./...`
